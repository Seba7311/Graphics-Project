# Vulkan Project

## Requirements
- Windows 10/11
- Visual Studio Community
- Vulkan SDK (LunarG)

## How to run
1. Install the Vulkan SDK from LunarG.
2. Creat an environment variable `VULKAN_SDK` that is set to the dir "C:\VulkanSDK\x.x.xxx.x".
3. Clone the repo and unzip.
4. Open `VulkanTutorial.sln` in Visual Studio.
5. Press F5 to Build + Run (x64, Debug or Release).

## Notes
- Shaders and textures are loaded from the `shaders/` and `texture/` folders.
